# Development

- You need install all requirements 

### 1. Clone Note Delivery Goods:

### Copy link

### 2. Open Note Delivery

### 3 Prepare to build Note Delivery Goods:
- Change appsettings.json
- Add Database in SQL Server
- Add Nuget Package 
- Add SQL Server file 
### 4. Build Product 
```bash
  cd Project
  dotnet run
```
